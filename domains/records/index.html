List domain records

GET /domains/:domain/records

Example

curl  -H 'X-DNSimple-Token: <email>:<token>' \
      -H 'Accept: application/json' \
      https://api.dnsimple.com/v1/domains/:domain/records

Parameters

Name Type Description
:domain string, integer The domain name or id
name string The name to search for. Example: www
type string The record type to search for. Example: www

Get a domain record

GET /domains/:domain/records/:id

Parameters

Name Type Description
:domain string, integer The domain name or id

Example

Get the record 123 for domain example.com.

curl  -H 'X-DNSimple-Token: <email>:<token>' \
      -H 'Accept: application/json' \
      https://api.dnsimple.com/v1/domains/example.com/records/123

Response

{
  "record": {
    "name": "www",
    "ttl": 3600,
    "created_at": "2010-07-01T08:01:18Z",
    "updated_at": "2010-10-21T15:47:47Z",
    "domain_id": 1,
    "id": 123,
    "content": "example.com",
    "record_type": "CNAME",
    "prio": null
  }
}

Create a domain record

POST /domains/:domain/records

Parameters

Name Type Description
:domain string, integer The domain name or id

Example

Create a record for domain example.com:

curl  -H 'X-DNSimple-Token: <email>:<token>' \
      -H 'Accept: application/json' \
      -H 'Content-Type: application/json' \
      -X POST \
      -d '<json>' \
      https://api.dnsimple.com/v1/domains/example.com/records

Input

record.name
Required string
record.record_type
Required string
record.content
Required string
record.ttl
Optional integer
record.prio
Optional integer 
{
  "record": {
    "name": "",
    "record_type": "MX",
    "content": "mail.example.com",
    "ttl": 3600,
    "prio": 10
  }
}

Response

Responds with HTTP 400 if bad request.

Responds with HTTP 400 if the validation fails.

Responds with HTTP 201 on success.

{
  "record": {
    "content": "mail.example.com",
    "created_at": "2013-01-29T14:25:38Z",
    "domain_id": 28,
    "id": 172,
    "name": "",
    "prio": 10,
    "record_type": "MX",
    "ttl": 3600,
    "updated_at": "2013-01-29T14:25:38Z"
  }
}

Update a domain record

PUT /domains/:domain/records/:id

Parameters

Name Type Description
:domain string, integer The domain name or id

Example

Update the record with ID 123 for domain example.com:

curl  -H 'X-DNSimple-Token: <email>:<token>' \
      -H 'Accept: application/json' \
      -H 'Content-Type: application/json' \
      -X PUT
      -d '<json>' \
      https://api.dnsimple.com/v1/domains/example.com/records/123

Input

The following fields are updateable.

record.name
Required string
record.content
Required string
record.ttl
Optional integer
record.prio
Optional integer 
{
  "record": {
    "content": "updated.example.com",
    "ttl": 600
  }
}

Response

Responds with HTTP 400 if bad request.

Responds with HTTP 400 if the validation fails.

Responds with HTTP 200 on success.

{
  "record": {
    "content": "updated.example.com",
    "created_at": "2013-01-29T14:25:38Z",
    "domain_id": 28,
    "id": 123,
    "name": "",
    "prio": 10,
    "record_type": "MX",
    "ttl": 600,
    "updated_at": "2013-01-29T14:25:38Z"
  }
}

Delete a domain record

Warning! There are planned changes for this API method. 
DELETE /domains/:domain/records/:id

Parameters

Name Type Description
:domain string, integer The domain name or id

Example

Delete the record with ID 123 for domain example.com:

curl  -H 'X-DNSimple-Token: <email>:<token>' \
      -H 'Accept: application/json' \
      -H 'Content-Type: application/json' \
      -X DELETE \
      https://api.dnsimple.com/v1/domains/example.com/records/123

Response

Responds with HTTP 200 on success.

Warning! The method will return a blank response in the future, you should not depend on the response body. See planned changes.
