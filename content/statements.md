# Statement API

API for listing account history as statements.

* TOC
{:toc}


## List statements

    GET /statements

### Example

    curl  -H 'X-DNSimple-Token: <email>:<token>' \
          -H 'Accept: application/json' \
          https://api.dnsimple.com/v1/statements
