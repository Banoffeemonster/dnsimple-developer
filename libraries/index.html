Libraries

There are several libraries available that use the DNSimple API. You can use these as a starting point or continue on through the documentation below.

If you’ve developed a library for the API, please let us know so we can add it here.

Please keep in mind that some of these libraries may be incomplete or oudated.

Ruby

  • DNSimple Ruby client
  • Fog - The Ruby cloud services library. Includes DNSimple support.

Python

  • dnsimple-python - A Python API client.

JavaScript

  • dnsimple-js - A jQuery-based DNSimple library.
  • nodejs-dnsimple - A Node.js API client.
  • dnsimple-cli - A command line tool written in Node.js

Golang

  • dnsimple-go - A Go API client.
  • domasimu - A simple domain management command line tool.

C♯

  • dnsimple-csharp - A C♯ API wrapper.

Java

  • dnsimple-java - A simple Java API wrapper.

PHP

  • dnsimple-php - A simple PHP API class.

PowerShell

  • PowerDNSimple - A PowerShell module for working with DNSimple API.

Also, make sure to check out the tools that are built on the DNSimple API.
