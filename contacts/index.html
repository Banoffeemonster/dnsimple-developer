Contact API

Contacts are used for domain registrant information when a domain is registered with DNSimple or transferred into DNSimple from another registrar.

  • List contacts
  • Get a contact
  • Create a contact
  • Update a contact
  • Delete a contact

List contacts

GET /contacts

Example

List all contacts.

curl  -H 'X-DNSimple-Token: <email>:<token>' \
      -H 'Accept: application/json' \
      https://dnsimple.com/contacts

Create a contact

POST /contacts

Example

Create a contact.

curl  -H 'X-DNSimple-Token: <email>:<token>' \
      -H 'Accept: application/json' \
      -H 'Content-Type: application/json' \
      -X POST \
      -d '<json>' \
      https://dnsimple.com/contacts

Input

contact.first_name
Required string
contact.last_name
Required string
contact.organization_name
Optional string representing the organization/company name. If the organization_name is specified, then you must also include job_title.
contact.job_title
Optional string
contact.address1
Required string
contact.city
Required string
contact.state_province
Required string
contact.postal_code
Required string
contact.country
Required string
contact.email_address
Required string
contact.phone
Required string
contact.phone_ext
Optional string
contact.fax
Optional string
contact.label
Optional string 
{
  "contact": {
    "first_name": "John",

    "phone": "505 111 2222",

    "job_title": "President",
    "label": "Office"

Update a contact

PUT /contacts/:id

Example

Update the contact with ID 123:

curl  -H 'X-DNSimple-Token: <email>:<token>' \
      -H 'Accept: application/json' \
      -H 'Content-Type: application/json' \
      -X PUT \
      -d '<json>' \
      https://dnsimple.com/contacts/123

Input

See Create a contact.

Response

Responds with HTTP 400 if the validation fails.

Responds with HTTP 200 on success.

Delete a contact

DELETE /contacts/:id

Example

Delete the contact with ID 123:

curl  -H 'X-DNSimple-Token: <email>:<token>' \
      -H 'Accept: application/json' \
      -H 'Content-Type: application/json' \
      -X DELETE \
      https://dnsimple.com/contacts/123

Response

Responds with HTTP 204 on success.
